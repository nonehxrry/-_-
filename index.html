Network-based Intrusion Detection System

Project Overview

This project involves developing a network-based intrusion detection system using Snort or Suricata. The system will set up rules and alerts to identify and respond to suspicious network activity and visualize detected attacks.

Tools and Technologies

  • Snort
  • Suricata
  • JavaScript
  • HTML & CSS

Setup Instructions

  1. Install Snort or Suricata on your system.
  2. Configure network interfaces and define rules.
  3. Set up alert mechanisms to notify suspicious activity.
  4. Use visualization tools to display detected attacks.
